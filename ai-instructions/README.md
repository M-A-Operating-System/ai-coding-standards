# AI Standards Knowledge Base

This folder contains the canonical standards, instructions, and templates for all AI-related work in this repository.

## Structure
- *.md: Root standards and master instructions
- languages/: Language-specific standards
- personas/: Persona definitions and routing rules
- skills/: Modular skills and process instructions
- prompt_templates/: Templates for prompt engineering and review
- data/: Data standards and schemas

## Change Control
All changes to standards require explicit approval and a summary of proposed changes. See master_standards.md for details.

## Quickstart
1. Review master_standards.md and persona_standards.md
2. Follow change control and forbidden patterns
3. Use skills/ for process guidance
