# Git Pipeline Operations & CI/CD Implementation ("How")

Aligns with: ops_standards.md

## Key Practices
- Use protected main branches and PR-based workflow.
- Implement CI pipeline steps: linting, testing, security scans, build artifacts.
- Automate deployment strategies (blue/green, canary) if applicable.
- Add health checks, smoke tests, and rollback steps to pipeline configs.
- Structured logging and metrics for pipeline runs.
