# Scripts Folder

This folder contains automation and sync scripts for Confluence, Figma, Git, and Jira integration.

## Key Scripts
- download_confluence.py / upload_confluence.py: Confluence sync
- download_figma.py: Figma asset sync
- download_gitActivity.py: Git activity sync
- download_jira.py / upload_jira.py: Jira sync
- initialize.py: Project initialization

See ai-instructions/skills/confluence_sync.md for workflow details.
